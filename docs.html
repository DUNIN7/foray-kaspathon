FORAY Protocol Documentation

FORAY Protocol is a privacy-preserving blockchain audit infrastructure designed to create immutable transaction records for enterprise systems. This documentation covers the v4.1 specification, integration patterns, and best practices.

4-Component Model

Every business transaction in FORAY is decomposed into four auditable components. This structure provides a complete trail from agreement to settlement.

Component Purpose Example
Arrangement Contractual foundation Master Service Agreement, Purchase Order
Accrual Formulas applied to asset flows Interest calculation, Depreciation expense, Overhead allocation
Anticipation Expected future flows Scheduled payment, Expected delivery
Action Actual asset transfers Wire transfer, Inventory receipt

Anticipations

  }

Privacy Architecture

FORAY implements an 8-layer privacy defense to protect sensitive business data while enabling cryptographic verification:

  1. Salted Formula IDs â€” Computation methods are hashed with entity-specific salts
  2. Multi-Layer Indirection â€” References use opaque identifiers
  3. Instance Pools â€” Similar transactions are batched to prevent pattern analysis
  4. Differential Privacy â€” Noise injection for aggregate statistics
  5. Tiered Oracle Attestation â€” Data verification without full disclosure
  6. Off-Chain Storage â€” Sensitive data never touches the blockchain
  7. Hash-Only Anchoring â€” Only cryptographic proofs are anchored
  8. Access Control â€” Role-based visibility for audit data

ERP Integration Patterns

FORAY is designed to work alongside existing enterprise systems, not replace them. The integration pattern follows these steps:

  1. Extract â€” Pull transaction data from ERP via API or export
  2. Transform â€” Map to FORAY 4-component model
  3. Hash â€” Generate component hashes and merkle root
  4. Anchor â€” Submit proof to Kaspa blockchain
  5. Store â€” Archive full transaction with anchor reference

FORAY adapters are available or planned for SAP, Oracle, NetSuite, and Salesforce. Contact us for enterprise integration support.
